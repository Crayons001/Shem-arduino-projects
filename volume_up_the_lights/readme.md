# Project Description

In this project, an LED bulb lights up depending on the intensity and duration of sound applied on a sound sensor. <br /><br />

Equipment used: <br />
-Arduino UNO <br />
-Sound sensor module (KY-038) <br />
-LED Bulb <br />
-220 Ohm resistor <br />
-Breadboard <br />
-Jumper wires <br /> <br />

Schematic is found in the file name 'shematic' <br />
